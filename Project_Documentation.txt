توثيق مشروع لعبة تخمين الرقم

المقدمة:

يهدف هذا المشروع إلى تصميم وتطوير لعبة تخمين رقم بسيطة باستخدام JavaFX. يقوم البرنامج باختيار رقم عشوائي بين 1 و 100، ويقوم اللاعب بمحاولة تخمين هذا الرقم من خلال إدخال أرقام والحصول على تلميحات إذا كان الرقم الذي تم تخمينه أكبر أو أصغر من الرقم الصحيح.

This project aims to design and develop a simple number guessing game using JavaFX. The program selects a random number between 1 and 100, and the player tries to guess this number by entering guesses and receiving hints if the guessed number is higher or lower than the correct number.

توضح هذه الوثيقة تفاصيل المشروع، من المفاهيم الأساسية إلى التصميم والتنفيذ، مع شرح مكونات النظام.

This document details the project from fundamental concepts to design and implementation, explaining system components.

Chapters:

1. أهداف المشروع
يهدف المشروع إلى:
- تطوير لعبة بسيطة لتخمين الأرقام لتعزيز مهارات المستخدم في التخمين والمنطق.
- بناء واجهة مستخدم رسومية بسيطة وسهلة الاستخدام باستخدام JavaFX.
- توفير تجربة تفاعلية مع رسائل تلميحية بعد كل محاولة.

1. Project Objectives
The project aims to:
- Develop a simple number guessing game to enhance user guessing and logic skills.
- Build a simple and user-friendly graphical interface using JavaFX.
- Provide an interactive experience with hint messages after each guess.

2. التقنيات المستخدمة
- لغة البرمجة: Java
- إطار العمل للواجهة: JavaFX
- البيئة التطويرية: Java Development Kit (JDK) وبيئة تطوير متكاملة مثل IntelliJ IDEA أو Eclipse.

2. Technologies Used
- Programming Language: Java
- UI Framework: JavaFX
- Development Environment: Java Development Kit (JDK) and an IDE such as IntelliJ IDEA or Eclipse.

3. وصف نظام اللعبة
واجهة المستخدم:
- حقل إدخال لكتابة رقم التخمين.
- زر لتأكيد التخمين.
- منطقة لعرض التلميحات والنتائج.
- زر لإعادة تشغيل اللعبة.

3. Game System Description
User Interface:
- Input field to enter guess number.
- Button to confirm the guess.
- Area to display hints and results.
- Button to restart the game.

آلية اللعبة:
- يتم اختيار رقم عشوائي بين 1 و 100 في بداية اللعبة.
- يدخل اللاعب رقم محاولته.
- يعرض البرنامج تلميحًا إذا كان الرقم الذي تم إدخاله أكبر أو أصغر من الرقم الصحيح.
- تنتهي اللعبة عند تخمين الرقم الصحيح مع عرض عدد المحاولات المستخدمة.

Game Mechanism:
- A random number between 1 and 100 is selected at game start.
- The player enters their guess.
- The program displays a hint if the guessed number is higher or lower than the correct number.
- The game ends upon guessing correctly, showing the number of attempts.

4. بنية النظام
- يتم تنفيذ اللعبة كتطبيق واجهة مستخدم واحدة (standalone).
- لا تعتمد على أي خادم أو نظام موزع.

4. System Architecture
- The game runs as a standalone GUI application.
- Does not rely on any server or distributed system.
